This file contains the main function and initialization logic for the Thale compiler. It handles command-line arguments, sets up the environment, and starts the compilation process.

The main entry point of the Thale compiler.

This function processes command-line arguments, opens the input file, reads its contents into memory, initializes the lexer, and retrieves tokens from the source code until the end of the file is reached.
